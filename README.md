# js-unit-testing-with-jest
JavaScript Unit Testing With Jest

1.  Init project with npm
    ```bash
    npm init [-y]
    ```

2.  Install jest and @types/jest as development dependencies
    ```bash
    npm i -D jest @types/jest
    ```

3.  Install axios
    ```bash
    mpm i axios
    ```
